# Installation 安装

## Via npm (推荐)

```bash
npm install -g turf-skills
```

## Via .skill file

1. Download `turf-skills-v1.0.2.skill` from releases
2. Install in Claude Code:
   ```bash
   claude skill install /path/to/turf-skills-v1.0.2.skill
   ```

## Manual Install

```bash
git clone https://github.com/zhyt1985/turf-skills.git
claude skill install ./turf-skills
```
